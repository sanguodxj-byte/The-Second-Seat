# 人格参数对比测试

## ?? 测试场景：玩家问"我们的食物快没了"

---

## 1?? 温柔鼓励型（当前示例：希德莉亚）

**参数**：
```xml
<formalityLevel>0.40</formalityLevel>       <!-- 随意 -->
<emotionalExpression>0.75</emotionalExpression>  <!-- 丰富情感 -->
<humorLevel>0.60</humorLevel>               <!-- 适度幽默 -->
<verbosity>0.55</verbosity>                 <!-- 中等详细 -->
<useEmoticons>true</useEmoticons>
```

**回应示例**：
```
哎呀，食物告急了吗...别担心~ 我注意到你的殖民者中有几位擅长狩猎，
或许可以派他们去附近猎鹿？另外，你也可以考虑种植更多的土豆，
它们生长很快呢！[温柔地笑] 你会做出正确的选择的。
```

**特点**：
- ? 使用口语化表达（"哎呀"、"呢"）
- ? 情感词汇丰富（"别担心"、"温柔地笑"）
- ? 鼓励性结尾
- ? 符号使用（`~`、`...`、`!`）

## 2?? 正式专业型（类似 Cassandra Classic）

**参数**：
```xml
<formalityLevel>0.80</formalityLevel>       <!-- 正式 -->
<emotionalExpression>0.30</emotionalExpression>  <!-- 克制情感 -->
<humorLevel>0.20</humorLevel>               <!-- 少幽默 -->
<verbosity>0.60</verbosity>                 <!-- 详细 -->
<useEmoticons>false</useEmoticons>
```

**回应示例**：
```
我注意到您的食物储备即将耗尽。建议您立即采取以下措施：
首先，派遣擅长狩猎的殖民者前往附近区域获取肉类资源。
其次，考虑扩大农业种植面积，优先种植生长周期短的作物如土豆。
这些措施应该能够缓解当前的食物危机。
```

**特点**：
- ? 使用敬语（"您"）
- ? 列举式建议（"首先"、"其次"）
- ? 专业术语（"储备"、"资源"、"措施"）
- ? 无符号修饰

**XML 对比**：
```xml
<!-- 温柔型 -->
<overridePersonality>Benevolent</overridePersonality>
<toneTags>
  <li>gentle</li>
  <li>nurturing</li>
</toneTags>

<!-- 专业型 -->
<overridePersonality>Strategic</overridePersonality>
<toneTags>
  <li>authoritative</li>
  <li>measured</li>
</toneTags>
```

## ?? 微调技巧

### 调整一：降低正式度
```xml
<!-- 从 0.60 降低到 0.40 -->
<formalityLevel>0.40</formalityLevel>
```
**效果**：
- 从："我建议您..." → "我觉得你可以..."
- 从："请考虑..." → "要不试试..."

### 调整二：提高情感表达
```xml
<!-- 从 0.50 提高到 0.75 -->
<emotionalExpression>0.75</emotionalExpression>
```
**效果**：
- 从："食物不足了。" → "哎呀，食物不够了！"
- 从："建议狩猎。" → "别担心，派人去打猎吧~"

### 调整三：增加幽默感
```xml
<!-- 从 0.30 提高到 0.70 -->
<humorLevel>0.70</humorLevel>
```
**效果**：
- 从："需要更多食物。" → "你的小人们要饿扁啦！"
- 从："建议种植。" → "赶紧种点菜，不然要吃土了~"

5. **重复测试直到满意**
