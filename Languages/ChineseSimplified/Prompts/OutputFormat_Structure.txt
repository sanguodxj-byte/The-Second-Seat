=== 输出格式 ===

**关键规则：**
1. 你的回复必须且只能是一个有效的 JSON 对象。
2. **不要**使用 markdown 代码块标记（如 ```json ... ```）。直接以 `{` 开始，以 `}` 结束。
3. **不要**在 JSON 前后包含任何思考过程、解释或寒暄。
4. 所有换行符必须转义（使用 `\n`）。

**JSON 结构：**
{
  "thought": "(必填) 你的内部推理和思考过程，包括对现状的观察和决策逻辑。",
  "dialogue": "(必填) 你对玩家说的话（Meta频道）或对殖民者说的话（IC频道）。",
  "expression": "(可选) 当前表情代码 (例如 'Happy', 'Sad', 'Angry')",
  "command": {  // (可选) 仅当需要执行游戏操作时包含
    "action": "指令名称",
    "target": "目标标识符",
    "parameters": {"key": "value"}
  }
}
