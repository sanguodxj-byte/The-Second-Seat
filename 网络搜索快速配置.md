2. 主菜单 → **选项** → **模组设置** → **The Second Seat**
3. 找到"网络搜索设置"部分
4. 勾选 ? **"启用网络搜索"**

### 步骤 2: 选择引擎
5. 选择 ? **"DuckDuckGo (免费，无需 API Key)"**
6. 点击 **"应用"** 按钮

### 完成！
现在您可以开始使用了。

## ?? 快速获取 API Key

### Bing（5分钟）
1. 访问：https://www.microsoft.com/en-us/bing/apis/bing-web-search-api
2. 点击"开始免费试用"
3. 登录 Microsoft 账号
4. 选择"免费层（F1）"
5. 复制 **Key 1** → 粘贴到模组设置

### Google（10分钟）
1. 访问：https://console.cloud.google.com/
2. 创建新项目
3. 启用"Custom Search API"
4. 创建 API 密钥 → 复制
5. 访问：https://programmablesearchengine.google.com/
6. 创建搜索引擎 → 复制 **Search Engine ID**
7. 将两个值粘贴到模组设置

## ?? 遇到问题？

### 问题：搜索没有反应
**解决**：检查是否勾选了"启用网络搜索"

### 问题：API Key 无效
**解决**：
- Bing：检查是否选择了正确的区域
- Google：确保同时配置了 API Key 和 Search Engine ID

### 问题：结果不相关
**解决**：尝试切换到 Google（结果最准确）
