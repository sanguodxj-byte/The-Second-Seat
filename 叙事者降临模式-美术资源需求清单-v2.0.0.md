## ? MVP 必需资源清单

### 1. 降临姿势立绘（3张）

#### 文件路径
```
Textures/UI/Narrators/Descent/Postures/Sideria/
├── descent_pose_ready.png
├── descent_pose_charging.png
└── descent_pose_casting.png
```

#### 规格要求
| 项目 | 要求 |
|------|------|
| 尺寸 | 1024 × 1574 像素 |
| 格式 | PNG（RGBA） |
| 色彩深度 | 32 位 |
| DPI | 300 |
| 文件大小 | 3-5 MB/张 |
| 总大小 | ~15 MB |

#### 内容描述
- **descent_pose_ready.png**：
  - 站立姿势，双手自然下垂
  - 平静的表情
  - 周围微弱光晕

- **descent_pose_charging.png**：
  - 单手抬起，掌心向上
  - 专注的表情
  - 手部聚集能量球（直径150-200px）
  - 衣物被魔力吹动

- **descent_pose_casting.png**：
  - 单手高举或双手向前推出
  - 凌厉的表情
  - 强烈光芒爆发
  - 魔法阵浮现（直径500-700px）

- 帧 13-15：快速下降
- 帧 16：减速，翅膀张开
- 帧 17：着地瞬间

**阶段 4：下马（帧 18-20）**
- 帧 18：叙事者准备下马

- **兼职制作**（4小时/天）：20 个工作日 = 4 周
