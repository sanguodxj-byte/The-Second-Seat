# 命令速查卡 - 批量俘虏与采矿

## ?? 批量俘虏 (BatchCapture)

### 快速使用
```
? "俘虏所有敌人"
? "抓捕倒地的人"
? "capture all enemies"
```

### 执行条件
| 条件 | 说明 |
|-----|------|
| ? 有倒地敌人 | 敌对、人类、未死亡 |
| ? 有看守 | 至少 1 名启用"看守"工作的殖民者 |
| ? 可到达 | 殖民者能到达敌人位置 |

### 常见反馈
```
? 成功：已指定俘虏 5 名敌人
? 失败：未找到可俘虏的倒地敌人
? 失败：无可用的看守执行俘虏任务
```

## ?? 常见问题

### Q: 俘虏不生效？
```
检查：
1. 工作优先级 → 启用"看守"
2. 建造监狱床
3. 确保殖民者不忙碌
```

### Q: 采集了不想要的矿？
```
使用更精确的目标：
? "只挖钢铁和银"
? "挖所有金属"（会包含玉石等）
```

### Q: AI 不理解指令？
```
使用明确关键词：
? "挖金属矿" / "俘虏敌人"
? "那个亮亮的东西" / "抓人"
```
