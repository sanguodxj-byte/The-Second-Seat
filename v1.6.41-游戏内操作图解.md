## ?? 常见问题排查

### **Q1：立绘看不见？**
```
检查清单：
□ 设置 → Mod选项 → The Second Seat → 勾选"使用立绘模式"
□ 人格已生成（需要先创建人格）
□ 游戏已完全加载（等待5秒）
```

### **Q2：按 Shift 没反应？**
```
检查清单：
□ 鼠标是否在立绘区域内（屏幕左侧）
□ Shift 键是否正常工作（测试其他游戏功能）
□ Tooltip 是否显示"?? 按住 Shift 键激活互动模式"
```

### **Q3：头部摸摸无法触发？**
```
检查清单：
□ 是否按住 Shift 键
□ 鼠标是否在头部区域（上方25%）
□ 是否进行了拖拽动作（不是单击）
□ 进度条是否达到60点
□ 冷却时间是否结束（3秒）
```

### **Q4：对话文本不显示？**
```
可能原因：
□ 好感度过低（低好感度对话较短）
□ 文本显示区域被其他UI遮挡
□ 浮动文本持续时间过短（2秒）

解决方法：
□ 尝试多次互动观察
□ 查看 DevMode 日志（F12 → 日志）
```
