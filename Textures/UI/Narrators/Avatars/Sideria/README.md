## ? 表情变体系统

### 工作原理

头像支持**随机表情变体**，与立绘系统完全同步：

```
切换到"开心"表情
    ↓
系统查找: happy.png, happy1.png, happy2.png...
    ↓
随机选择一个存在的文件
    ↓
显示选中的头像变体
```

### 变体命名规范

```
{expression}.png       → 基础版本
{expression}1.png      → 变体1
{expression}2.png      → 变体2
{expression}3.png      → 变体3
{expression}4.png      → 变体4
{expression}5.png      → 变体5
```

### 示例

```
happy.png     → 微笑（基础）
happy1.png    → 眯眼笑（变体1）
happy2.png    → 开怀大笑（变体2）
happy3.png    → 抿嘴笑（变体3）
```

**效果**：每次切换到"开心"表情时，从这4个中随机选一个！

## ?? 规格要求

- **尺寸**: 512x512 像素
- **格式**: PNG
- **背景**: 建议透明
- **内容**: 头部特写（面部清晰）
- **命名**: 全小写，如 `happy.png`
- **变体**: 可选，最多 5 个（基础 + 变体1-5）

## ? 新增：害羞表情

### 设计要点

**基础害羞 (`shy.png`)**：
- 微红的脸颊
- 眼神略微飘移
- 嘴角微微翘起
- 整体温柔的感觉

**害羞变体1 (`shy1.png`)**（可选）：
- 双手捂脸
- 眼睛眯起
- 脸更红

**害羞变体2 (`shy2.png`)**（可选）：
- 低头害羞
- 侧目偷看
- 轻咬嘴唇

### 触发场景

- 玩家感谢叙事者
- 对话包含害羞关键词
- 好感度达到特定阶段
- TTS 使用温柔语气（gentle）

## ?? 设计建议

### 构图比例

```
┌─────────────┐
│             │  ← 顶部留白 10%
│   ?? 面部    │  ← 面部特写 80%
│             │  
│   肩部      │  ← 底部留白 10%
└─────────────┘
```

### 表情重点

1. **眼睛**：最重要的情感表达区域
2. **嘴巴**：辅助表达情绪
3. **脸颊**：可添加脸红、皱纹等细节
4. **整体氛围**：通过光影、色调营造情绪

### 变体差异建议

不同的变体可以有：
- **角度差异**：正面、侧面、微侧
- **强度差异**：轻微开心 vs 大笑
- **细节差异**：眼睛睁开程度、嘴形

## ?? 与立绘同步

- 变体选择**同步随机**（使用相同算法）
- 文件命名规范**完全一致**
- 触发条件**保持一致**
