# 头像文件夹 (Avatars)

此文件夹存放叙事者头像（512x512 像素）

## 命名规范

### 基础头像
- {PersonaName}.png - 默认头像
- {PersonaName}_neutral.png - 中性表情头像

### 表情头像（可选）
- {PersonaName}_happy.png - 开心
- {PersonaName}_sad.png - 悲伤
- {PersonaName}_angry.png - 愤怒
- {PersonaName}_surprised.png - 惊讶

## 示例
`
Cassandra.png
Phoebe.png
Randy.png
`

## 图片规格
- 尺寸: 512x512 像素
- 格式: PNG（支持透明）
- 推荐: 使用圆形或方形头像
