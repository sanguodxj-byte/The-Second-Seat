# UI Textures

## NarratorButton.png

**Purpose**: Icon for the AI Narrator screen button

### Specifications
- **Size**: 48x48 pixels (recommended)
- **Format**: PNG with transparency
- **Location**: This folder (`Textures/UI/NarratorButton.png`)

### Design Guidelines
- Transparent background
- Simple, clear iconography
- Consistent with RimWorld UI style
- Suggestions: Chat bubble, microphone, AI symbol, or book icon

### If Missing
The mod will use a fallback icon (RimWorld's built-in info icon) if this file doesn't exist.

---

## Quick Start

1. Design a 48x48 PNG icon
2. Name it `NarratorButton.png`
3. Place it in this folder
4. Redeploy the mod
5. Restart RimWorld to see your custom icon!

For detailed design guidelines, see: `按钮图标设计指南.md`
