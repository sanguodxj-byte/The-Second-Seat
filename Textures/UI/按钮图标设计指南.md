## ?? 图标规格

### 尺寸要求
- **推荐尺寸**：`48x48 像素`
- **最小尺寸**：`32x32 像素`
- **最大尺寸**：`64x64 像素`

### 文件格式
- **格式**：PNG（支持透明度）
- **颜色**：RGB 或 RGBA
- **透明度**：支持 Alpha 通道

### 设计要求
- **背景**：透明背景（推荐）
- **边距**：图标周围留 4-8px 空白
- **风格**：简洁、清晰、易识别
- **颜色**：与 RimWorld UI 风格一致

## ?? 如果没有自定义图标

### 备用图标
如果 `NarratorButton.png` 不存在，系统会使用：
- **默认图标**：RimWorld 内置的信息图标（TexButton.Info）
- **功能**：完全正常工作
- **外观**：标准的 "i" 信息图标

### 创建临时图标
可以先使用简单的纯色方块测试：

```powershell
# 使用 PowerShell 创建简单测试图标（需要 .NET 库）
Add-Type -AssemblyName System.Drawing
$bmp = New-Object System.Drawing.Bitmap(48, 48)
$g = [System.Drawing.Graphics]::FromImage($bmp)
$brush = New-Object System.Drawing.SolidBrush([System.Drawing.Color]::LightGray)
$g.FillEllipse($brush, 4, 4, 40, 40)
$bmp.Save("Textures\UI\NarratorButton.png")
$g.Dispose()
$bmp.Dispose()
Write-Host "? 已创建测试图标" -ForegroundColor Green
```

## ?? 技术支持

### 如果按钮不显示
1. 检查 MapComponent 是否正确注册
2. 查看 Player.log 中的错误
3. 确认游戏处于 Playing 状态
4. 验证 Defs 文件正确

### 如果图标不显示
1. 检查文件路径拼写
2. 确认 PNG 格式正确
3. 尝试删除图标使用备用
4. 查看日志中的纹理加载错误
