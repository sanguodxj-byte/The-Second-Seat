# 立绘模式 - 快速参考

## ?? 立绘模式

**版本：** v1.6.16  
**功能：** 支持 1024x1572 全身立绘

---

## ?? 快速启用

### 设置路径
```
主菜单 → 选项 → Mod 设置 → The Second Seat
```

### 启用选项
```
[?] 使用立绘模式（1024x1572 全身立绘）
```

---

## ?? 模式对比

| 模式 | 尺寸 | 内容 |
|------|------|------|
| **头像** | 512x512 | 脸部特写 |
| **立绘** | 1024x1572 | 全身立绘 |

---

## ?? 文件位置

### 头像（512x512）
```
Textures/UI/Narrators/Avatars/
└── {Name}/base.png
```

### 立绘（1024x1572）
```
Textures/UI/Narrators/9x16/
└── {Name}/base.png
```
