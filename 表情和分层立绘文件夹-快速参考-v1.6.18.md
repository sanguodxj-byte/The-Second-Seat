## ?? 快速要点

### **表情文件夹（Expressions）**
- **用途：** 存放完整表情立绘或表情专用层
- **命名格式：** `{expression}.png`（全小写）
- **支持变体：** `happy1.png`, `happy2.png`, etc.
- **回退机制：** 变体 → 基础 → 默认 → 占位符

### **分层立绘文件夹（Layered）**
- **用途：** 存放分层立绘纹理（身体、表情、眼睛、嘴巴等）
- **必需文件：** `base_body.png`, `face_neutral.png`
- **动画层：** `eyes_*.png`（眨眼），`mouth_*.png`（张嘴）
- **图层合成：** 按优先级自动叠加

## ?? 文件命名快查

### **表情文件夹**
```
Expressions/{PersonaName}/
├── neutral.png          ← 默认表情（必需）
├── happy.png            ← 基础开心
├── happy1.png           ← 开心变体1
├── sad.png              ← 悲伤
├── angry.png            ← 生气
├── surprised.png        ← 惊讶
├── confused.png         ← 疑惑
├── smug.png             ← 得意
└── shy.png              ← 害羞
```

### **分层立绘文件夹**
```
Layered/{PersonaName}/
├── base_body.png        ← 身体层（必需）
├── face_neutral.png     ← 默认表情（必需）
├── eyes_open.png        ← 睁眼（眨眼动画）
├── eyes_half.png        ← 半闭眼（眨眼动画）
├── eyes_closed.png      ← 闭眼（眨眼动画）
├── mouth_closed.png     ← 闭嘴（张嘴动画）
├── mouth_smile.png      ← 微笑（张嘴动画）
├── mouth_open_small.png ← 小张嘴（张嘴动画）
└── mouth_open_wide.png  ← 大张嘴（张嘴动画）
```

## ?? 纹理规格统一要求

| 属性 | 要求 |
|------|------|
| 尺寸 | 1024×1574 像素 |
| 格式 | PNG（透明通道） |
| 背景 | 完全透明 |
| 分辨率 | 300 DPI（推荐） |
| 文件大小 | < 2MB |
| 命名规范 | 全小写、无空格、无特殊字符 |

---

## ?? 图层对齐原则

> **所有动画层（eyes_*, mouth_*）必须与 `face_neutral.png` 像素级对齐！**

### **对齐检查清单**
- [ ] 所有文件尺寸精确 1024×1574
- [ ] 眼睛层与 `face_neutral.png` 对齐
- [ ] 嘴巴层与 `face_neutral.png` 对齐
- [ ] 表情层面部位置一致
- [ ] 服装层与 `base_body.png` 对齐

## ?? 动画效果

### **眨眼动画（3-6秒周期）**
```
eyes_open → eyes_half → eyes_closed → eyes_half → eyes_open
```

### **张嘴动画（TTS 同步）**
```
播放 TTS：mouth_closed ? mouth_open_small/wide
停止 TTS：mouth_closed
```
